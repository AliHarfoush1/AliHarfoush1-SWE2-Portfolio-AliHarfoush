# Task: Vehicle System

## What I Built
A Java program with 3 types of vehicles: Car, Motorcycle, and Truck.

## Classes
- **Vehicle** (abstract) - parent class
- **move** (interface) - for movement
- **Car, Motorcycle, Truck** - the actual vehicles

## What Each Does
- **Car:** diesel, shut windows
- **Motorcycle:** check helmet
- **Truck:** carry cargo

## Goal
Practice OOP basics.
